# Property Management System

A web-based application for managing properties, users, and maintenance requests.

## Overview

This system allows administrators and users to manage different types of properties such as apartments, villas, and lands. It includes features for listing, viewing, and managing property details along with user interactions.

## Features

* Admin dashboard for managing properties
* Add, view, and manage apartments, villas, and lands
* User registration and login system
* Property listing and detailed view pages
* Maintenance request handling
* Agent and property management modules

## Project Structure

* `app.py` – Main application file
* `templates/` – HTML pages
* `static/` – CSS and static assets
* `Pictures/` – Image resources

## Tech Stack

* Python (Flask)
* HTML, CSS
* SQLite (or local database)
